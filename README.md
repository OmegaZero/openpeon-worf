# What is this

This is a sound pack for [Peon-Ping](https://github.com/PeonPing/peon-ping), a sound overlay for claude.

It adds sounds from WarCraft 3 pertaining to [Jaina Proudmore](https://wowpedia.fandom.com/wiki/Jaina_Proudmoore). All sound files are from the game and were pulled from [D2 mod - PhrozenKeep](https://d2mods.info/forum/downloadsystemcat?id=52&start=15) at [this link](https://phrozenkeep.blob.core.windows.net/public/files/plugins/audio/sfx/wc3sfx-human-jaina.zip).
