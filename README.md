# What is this

This is a sound pack for [Peon-Ping](https://github.com/PeonPing/peon-ping), a sound overlay for claude.

It adds sounds from WarCraft 3 pertaining to [Worf](https://memory-alpha.fandom.com/wiki/Worf). All sound files are from various Star Trek series and movies the character was in. Clips were pulled from sound bytes around the internet.
